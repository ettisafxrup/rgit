rgit v1.0

Welcome to the rgit installer!

rgit is a handy command-line tool that simplifies and shortens common Git operations.

With rgit, you can quickly push, pull, rename branches, and more, all with just straight-forward simple commands.

Before proceeding, please ensure that you have Git installed on your machine and accessible from your terminal.

For detailed documentation and support, visit:
(It is an Open Source project, you can feel free to contribute anyday!)
